# financial-ratio-analyzer

기업 유형에 따라 적절한 재무비율 체계를 자동으로 선택하는 분석 도구입니다.

## Background

2024년 11월 재무제표 분석 과제에서 메리츠금융지주를 분석하며, 일반 기업에
통상적으로 쓰이는 차입금비율·이자보상비율·고정장기적합율이 금융지주회사
분석에는 적합하지 않다는 판단을 내렸습니다. 금융지주회사는 부채비율과
이중레버리지비율에 대해 금융당국의 직접적인 규제를 받기 때문에, 이 두
지표가 실질적인 재무위험을 더 정확히 반영한다고 보았기 때문입니다.

이 프로젝트는 **2024년 11월 시점의 시황 분석이나 투자 판단을 다시 다루는
것이 아니라**, 당시 내렸던 "기업 유형에 따라 적절한 지표를 선택한다"는
판단 논리 자체를 코드로 재구성한 것입니다. 데이터는 그 로직을 검증하기
위한 실제 사례로만 사용했습니다.

## Goal

- 기업이 금융지주회사인지 여부에 따라 적용할 재무비율 체계를 자동으로 분기
- 금융지주회사인 경우, 이중레버리지비율이 금융당국 규제선(130%) 대비
  얼마나 여유가 있는지 자동 판정
- 표준 재무비율(차입금비율, 이자보상비율)과 규제 기준 비율(부채비율,
  이중레버리지비율) 중 어떤 것이 왜 적용됐는지 판단 근거를 함께 출력

## How it works

### 1. 기업 유형 판정 (`FinancialStatement.is_financial_holding`)

기업이 금융지주회사인지 여부를 입력받습니다. 이 값에 따라 `analyze()`
함수가 서로 다른 비율 체계를 적용합니다.

### 2. 비율 선택 및 계산 (`analyze`)

- **금융지주회사**: 부채비율(부채총계/자본총계) + 이중레버리지비율(자회사
  출자총액/자본총계 x 100)을 계산하고, 규제선(130%) 대비 여력을 함께 산출
- **일반 기업**: 차입금비율 + 이자보상비율을 계산 (해당 데이터가 있는 경우)

각 결과에는 왜 그 비율 체계를 선택했는지 판단 근거(`reason`)가 함께
출력됩니다.

### 실행 예시

```bash
python financial_ratio_analyzer.py
```

## Why this matters

동일한 "재무 건전성 분석"이라는 목적이라도, 기업의 구조적 특성(지주회사
여부)에 따라 적절한 지표가 달라질 수 있다는 것을 재무제표 분석 과제를
통해 직접 경험했습니다. 이 판단을 코드로 구조화하면서, 표준화된 접근이
항상 정답은 아니며 대상의 특성을 먼저 파악한 뒤 분석 방법을 선택하는
과정이 중요하다는 점을 다시 확인했습니다.
